2. Arrange, Act, Assert

Structure for a testing environment.
Arrange – Set up the test by preparing any necessary objects, dependencies, or initial conditions.
Act – Execute the actual behavior or function being tested.
Assert – Verify the outcome by checking if the expected result matches the actual result.
